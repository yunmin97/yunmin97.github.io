Cesium for Unreal

2021-07-04 — Written by Yunmin

1、描述：
  Cesium for Unreal插件在UE4引擎中解锁了三维地理空间生态系统，通过结合一高精度全尺寸WGS84地球仪、开放的api和空间索引(如3D Tiles)的开放标准，以及利用虚幻引擎的力量实现的基于云端的真实3D内容，您将有权自主通过使用游戏引擎和利用真实世界的内容创建地理空间应用的新时代。
  Cesium for Unreal插件可以把真实的3D内容（如：高分辨率摄影测量、地形、图像、来自Cesium ion的3D建筑和其他资源）转化为数据流。插件集成了Cesium ion，可以一键访问全球3D内容的流数据。Ion用户还可以利用基于云的3D贴片管道来创建端到端工作流，将大量3D内容转换为语义丰富的3D Tiles。
  插件支持基于开放标准和api的云端和私有网络内容和服务。您可以自由地使用任何源代码与插件的组合。
2、特性：

  • 使用开放标准和api，在虚幻引擎中实现高精度的全尺寸地球仪，以高精度创建真实世界的数字化双胞胎。
  • 使用3D Tiles空间索引为高分辨率摄影测量、地形、图像和3D城市实现可视化真实世界的3D内容，以创建丰富的环境。
  • Cesium ion集成一键式访问全球策划的3D内容。
  • 集成了虚幻引擎编辑器、Actors和组件、蓝图、物理、碰撞、地编和植被系统，和其他UE4功能。

3、技术细节：

  • Cesium编辑器：使用Cesium编辑器创建数字化真实世界应用程序，包括集成了Cesium ion，可把全球3D内容以数据流形式输送到虚幻引擎。
  • Cesium运行时：Cesium运行时模块运行你发布应用程序时不需要打包所有内容，而是使用数据流来创造全球体验。

4、平台支持：
  Windows 10 x64, MacOS, Linux
5、许可：
  Apache 2.0，可免费用于商业和非商业用途
